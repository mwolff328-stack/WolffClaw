## DESIGN.md Evaluation (Deb)

### Task
Mike wanted to adopt a DESIGN.md from the VoltAgent/awesome-design-md collection for SurvivorPulse's UI design system.

### Deb's Shortlist (5 of 58 reviewed)
1. **Linear** ⭐ (Deb's pick) — dark-mode-native, Inter Variable, luminance stacking, Radix UI compatible
2. **IBM Carbon** — enterprise analytics gold standard, IBM Plex Sans, rigorous token system
3. **Sentry** — most analogous product context, deep purple-black, uppercase label system
4. **Vercel** — precision minimalism, Geist with tabular numbers, shadow-as-border
5. **PostHog** — exact same tech stack (Tailwind + Radix + shadcn), wrong aesthetic but right bones

### Decision
Mike accepted Deb's recommendation: **Linear as base**, with IBM Carbon token naming for status colors and Sentry's uppercase label pattern.
