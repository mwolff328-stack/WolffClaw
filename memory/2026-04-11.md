### Major Pivot: Independent Path
- Hid coordinated path (Moments 2-3, Season Replay Table, Correlation Risk) because coordinated approach was performing worse than independent
- Rewired pick logic to use full 32-team games API instead of 5-entry allocation array
- Independent path picks by highest win probability per entry, no team reuse within week or across weeks for same entry

### Bug Fixes
- Week 1 API timeout (increased to 90s)
- Available Teams showing only 5 teams (was using allocation[] not games API)
- Games API only returning Week 1 (needed week=N parameter per request)
- Parallel game fetch with retry (was sequential, weeks could silently fail)
- Win probability conversion formula fixed (spread_line convention was inverted)

### Strategy Research (Stan)
- **Round 1**: Tested 4 strategies against 2025 data. Pure Win Probability best (Week 10). Leverage/contrarian strategies crashed (picked unsafe teams).
- **Round 2**: Tested 7 constrained strategies. **70/30 Weighted Blend won** (Week 14 vs Week 10 for baseline). Formula: `0.7 * winProb + 0.3 * (1 - pickShare/100)`
- **Multi-Season**: Tested all 7 across 2023, 2024, 2025. **70/30 Blend confirmed winner** (57 total entry-weeks vs 38 for pure win prob, 50% improvement). Best or tied-best in every season.
